# react-hn
## Hacker News client made via React


### Quick start
Clone the repo:
```bash
git clone https://github.com/moondef/react-hn.git && cd react-hn
```

Install deps:
```bash
npm install
```

#### Scripts
| `npm run <script>` | Description                  |
| ------------------ | ---------------------------- |
| `start`            | Runs app on `localhost:3000` |
| `build`            | Builds app to ./build        |
